# 🔥 Calorie Calculator (BMR-Based)

## 📌 Overview
This project is a command-line calorie calculator that estimates daily calorie needs based on user inputs such as weight, height, age, and activity level.

It uses the BMR (Basal Metabolic Rate) formula combined with activity multipliers to calculate maintenance calories and fitness goals.

---

## 🧠 Objective
To provide a simple and practical tool for estimating daily calorie requirements for fitness planning.

---

## ⚙️ How It Works

1. Calculates BMR using:
   - Weight
   - Height
   - Age

2. Applies activity multiplier based on workout frequency

3. Outputs:
   - Maintenance calories
   - Fat loss calories
   - Muscle gain calories

---

## 📊 Formula Used

BMR (Mifflin-St Jeor Equation):
- `BMR = 10 × weight + 6.25 × height - 5 × age + 5`
